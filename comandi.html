Lista Comandi

In questa pagina trovi tutti i comandi personalizzati disponibili nel server NerdPlayers. Alcuni comandi potrebbero richiedere permessi specifici per funzionare correttamente.

Comandi Generali

  • /spawn - Teletrasportati allo spawn principale.
  • /sethome - Imposta la tua posizione attuale come home.
  • /home - Teletrasportati alla tua home.
  • /delhome - Rimuovi la tua home.
  • /tpa [giocatore] - Invia una richiesta di teletrasporto a un altro giocatore.
  • /tpaccept - Accetta una richiesta di teletrasporto.
  • /tpdeny - Rifiuta una richiesta di teletrasporto.

Comandi Economy

  • /balance - Mostra il tuo saldo attuale.
  • /pay [giocatore] [importo] - Invia soldi a un altro giocatore.
  • /shop - Apre il negozio in-game.
  • /sell - Vendi gli oggetti tenuti in mano o nella GUI.

Comandi Utility

  • /rules - Mostra le regole del server.
  • /discord - Ottieni il link per entrare nel server Discord.
  • /vote - Ricevi i link per votare il server.
  • /msg [giocatore] [messaggio] - Invia un messaggio privato a un altro giocatore.

Altri Comandi Utili

  • /back - Torna alla tua posizione precedente (dopo un death o teletrasporto).
  • /warp [nome] - Teletrasportati a un warp predefinito.
  • /kit - Visualizza o richiedi kit disponibili (se presenti).
